  
**Diagnóstico de marca**  
Alza Construcciones

Cuestionario de entrevista en profundidad  
BARCO Estrategia de Marca  ·  Arquitectura comercial B2B  ·  8 bloques  ·  46 preguntas

| 0 | Datos de referencia y contexto de la entrevista Información operativa básica para encuadrar el diagnóstico. |
| :---: | :---- |

| 0.1  ¿Quién o quiénes responden este cuestionario? ¿Cuál es su rol dentro de Alza? |
| :---- |

| 0.2  ¿Cuántas personas integran hoy el equipo? ¿Cómo está estructurado organizacionalmente? |
| :---- |

| 0.3  ¿Existen socios, directores o perfiles fundadores cuya visión sea central para esta conversación y que no estén presentes? |
| :---- |

| 1 | Historia y origen Entender la narrativa fundacional, los hitos y cómo la empresa llegó a ser lo que es. |
| :---: | :---- |

| 1.1  ¿Cómo surgió Alza? ¿Cuál fue el problema o la oportunidad que llevó a sus fundadores a crear la empresa en 2015? |
| :---- |

| 1.2  ¿Hubo un proyecto o cliente particular que marcó el punto de inflexión de la empresa? ¿Cuál fue y por qué lo consideran así? |
| :---- |

| 1.3  ¿De dónde viene el nombre "Alza"? ¿Tiene un significado deliberado o una historia detrás? |
| :---- |

| 1.4  Si tuviesen que describir los tres momentos más importantes en la historia de la empresa hasta hoy, ¿cuáles serían? |
| :---- |

| 1.5  ¿Cómo fue evolucionando el foco del negocio desde el inicio? ¿Siempre se orientaron a arquitectura comercial y corporativa o hubo otras etapas? |
| :---- |

| 2 | Identidad de marca actual Cómo se perciben, qué creen que proyectan y qué distancias existen entre identidad y percepción real. |
| :---: | :---- |

| 2.1  Si tuviesen que definir Alza en tres palabras —no lo que hacen, sino quiénes son— ¿qué tres palabras elegirían? |
| :---- |

| 2.2  ¿Tienen hoy una misión, visión y/o valores formalizados? ¿Están escritos? ¿Los usa el equipo cotidianamente? |
| :---- |

| 2.3  ¿Cómo creen que sus clientes actuales describirían a Alza si les preguntáramos? ¿Coincide con cómo se perciben ustedes? |
| :---- |

| 2.4  ¿Existe algún aspecto de la empresa que sientan que no está comunicado ni reconocido suficientemente por el mercado? |
| :---- |

| 2.5  ¿Tienen un perfil más asociado al diseño/creatividad o a la ejecución/ingeniería? ¿Es una tensión que sienten o han resuelto? |
| :---- |

| 2.6  ¿Cómo describirían el «carácter» de Alza como empresa? ¿Qué adjetivos le pondrían si fuera una persona? |
| :---- |

| 3 | Propuesta de valor y diferenciación Qué ofrecen realmente, qué los diferencia y qué promesa sostiene el negocio. |
| :---: | :---- |

| 3.1  ¿Qué es lo que Alza hace que ningún otro estudio o empresa de construcción comercial en Córdoba hace igual? ¿Cuál es su ventaja real? |
| :---- |

| 3.2  ¿Qué problemas concretos resuelven a sus clientes que van más allá del «construir bien»? |
| :---- |

| 3.3  Su web menciona que los espacios que proyectan «transmiten con eficacia los valores de cada compañía». ¿Pueden dar un ejemplo de un proyecto donde eso haya sido especialmente logrado? *↳ Anclaje en la evidencia: casos de portafolio como Naranja X, Sura, Batistella* |
| :---- |

| 3.4  ¿Cómo definen la relación entre diseño y construcción dentro de su propuesta? ¿Ofrecen ambas como servicio integrado o las manejan por separado según el cliente? |
| :---- |

| 3.5  ¿Qué es lo que un cliente pierde si no trabaja con Alza y elige otra opción del mercado? |
| :---- |

| 3.6  ¿Tienen una política diferenciada en términos de sustentabilidad o impacto ambiental? ¿Es un diferenciador real o más aspiracional hoy? |
| :---- |

| 4 | Posicionamiento y competencia Dónde está Alza en el mapa competitivo, cómo se percibe y hacia dónde quiere ir. |
| :---: | :---- |

| 4.1  ¿Quiénes son sus principales competidores directos? ¿Con quiénes se disputan proyectos habitualmente? |
| :---- |

| 4.2  ¿Con qué competidor o referente del sector —dentro o fuera de Argentina— se sienten más identificados en términos de perfil o aspiración de marca? |
| :---- |

| 4.3  ¿Cómo perciben que los ven los clientes frente a esa competencia? ¿Más caros, más creativos, más confiables, más ágiles? |
| :---- |

| 4.4  ¿Hay proyectos que han perdido frente a competidores? ¿Saben por qué? ¿Qué aprendieron de eso? |
| :---- |

| 4.5  ¿Cuál es el segmento o tipo de proyecto donde Alza gana con mayor consistencia? ¿Y en cuál sienten que no están compitiendo aún pero podrían? |
| :---- |

| 5 | Audiencias y relaciones comerciales A quién le habla Alza, cómo llegan los clientes y qué vínculos sostienen el negocio. |
| :---: | :---- |

| 5.1  ¿Cómo es el perfil de su cliente ideal? ¿Hay diferencias entre el cliente que tienen hoy y el que querrían tener? |
| :---- |

| 5.2  ¿Cómo llegan los clientes actualmente? ¿Por referidos, por presencia digital, por licitaciones, por redes de contacto? |
| :---- |

| 5.3  ¿Con quién dentro de las empresas clientes suelen tratar? ¿Con áreas de real estate, con marketing, con directivos? *↳ Relevante para definir el tono y el foco comunicacional en B2B* |
| :---- |

| 5.4  ¿Qué porcentaje de su cartera es de clientes recurrentes versus clientes nuevos? ¿Hay fidelidad? ¿Por qué creen que vuelven o no vuelven? |
| :---- |

| 5.5  ¿Tienen proveedores, contratistas o aliados estratégicos que sean parte esencial de cómo Alza opera y entrega valor? |
| :---- |

| 6 | Comunicación y presencia de marca Cómo comunican hoy, qué funciona, qué no y dónde están los vacíos. |
| :---: | :---- |

| 6.1  ¿Cómo describirían su presencia de marca actual? ¿Sienten que comunican bien quiénes son o que su identidad está por debajo de lo que la empresa realmente es? |
| :---- |

| 6.2  ¿Qué canales utilizan actualmente para comunicar? ¿Cuál sienten que tiene mejor retorno? |
| :---- |

| 6.3  ¿Tienen un manual de identidad o guía de estilo visual definida? ¿Se aplica de forma consistente? |
| :---- |

| 6.4  ¿Tienen alguna forma de sistematizar o comunicar sus casos de éxito? ¿El portafolio trabaja activamente como herramienta comercial? |
| :---- |

| 6.5  Si tuviesen que evaluar su presencia digital hoy del 1 al 10, ¿qué número pondrían? ¿Por qué? |
| :---- |

| 6.6  ¿Hay alguna empresa del sector —local, nacional o internacional— cuya comunicación admiren o les sirva de referencia? |
| :---- |

| 7 | Cultura organizacional y equipo La identidad hacia adentro: cómo trabajan, qué valoran y cómo viven la marca. |
| :---: | :---- |

| 7.1  ¿Cómo describirían la cultura interna de Alza? ¿Qué hace que alguien «encaje» o no en el equipo? |
| :---- |

| 7.2  ¿El equipo conoce y comparte los valores de la empresa? ¿O esos valores viven más en los fundadores/líderes? |
| :---- |

| 7.3  ¿Hay roles o perfiles dentro del equipo que consideren clave para sostener la propuesta de valor de Alza? |
| :---- |

| 7.4  ¿Cómo es la dinámica entre arquitectura/diseño y ejecución/obra dentro del equipo? ¿Conviven bien esos dos mundos? |
| :---- |

| 8 | Proyección estratégica Hacia dónde va Alza y qué rol debería jugar la marca en ese camino. |
| :---: | :---- |

| 8.1  ¿Dónde imaginan a Alza en 3 a 5 años? ¿En tamaño, en tipo de proyectos, en geografía? |
| :---- |

| 8.2  ¿Hay alguna línea de servicio o nicho que quieran desarrollar y que hoy no estén explotando? |
| :---- |

| 8.3  ¿Qué frena hoy el crecimiento de Alza? ¿Es un problema de capacidad, de posicionamiento, de acceso a nuevos clientes, de equipo? |
| :---- |

| 8.4  ¿La marca actual «aguanta» ese crecimiento o es uno de los factores que hay que actualizar para escalar? |
| :---- |

| 8.5  Si hubiera una sola cosa que este proceso de estrategia de marca debería resolver para Alza, ¿qué sería? |
| :---- |
